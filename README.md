# UTS IoT - Back-End Python

**Nama:** Raelqiansyah Putranta Dibrata

Project ini adalah back-end untuk tugas UTS Pemrograman IoT.  
Project menggunakan Python dan MQTT untuk komunikasi dengan perangkat IoT.  

---

## Keterangan File
| File | Keterangan |
|------|-----------|
| `app.py` | File utama untuk menjalankan server dan API |
| `mqtt_subs.py` | File untuk subscribe dan handle data MQTT |
| `requirements.txt` | List library Python yang dibutuhkan |
| `venv/` | Virtual environment (tidak diikutkan di GitHub) |

---

## Fitur
- Meng-handle koneksi MQTT
- Menyimpan data sensor
- Endpoint API sederhana
